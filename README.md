# Elio — AI Recipe Generator

> "Already knows your kitchen."

Elio is a Flutter app that generates personalised recipes using Gemini 2.0 Flash. It knows your pantry staples, dietary requirements, and household preferences — so you only need to tell it what's fresh today.

---

## Sprints

| Tag | Description |
|-----|-------------|
| `sprint-1` | Onboarding flow (screens 0–4), Firebase Auth, Firestore, design system |
| `sprint-2` | Style preferences (screen 5), home screen with mood chips, Gemini recipe generation, recipe display screen, daily generation cap |

---

## Setup

### Prerequisites
- Flutter 3.27.4
- Android SDK with NDK 27.0.12077973
- Firebase project with Android app registered

### Firebase credentials
1. Download `google-services.json` from your Firebase console and place it in `android/app/`
2. Run `flutterfire configure` to generate `lib/firebase_options.dart`

### Build
```bash
flutter pub get
flutter build apk --debug
```

## Architecture

```
lib/
  main.dart                  # Entry point, auth gate
  firebase_options.dart      # Generated by flutterfire (not committed)
  theme/
    elio_theme.dart          # Design system: colours, typography, ThemeData
  models/
    elio_models.dart         # Core domain models
    onboarding_state.dart    # Onboarding data model
    recipe_models.dart       # Recipe request/response models
  services/
    auth_service.dart        # Firebase Auth + Google Sign-In
    firestore_service.dart   # Firestore read/write
    gemini_service.dart      # Gemini 2.0 Flash API
  screens/
    onboarding/              # Screens 0–5
    home/                    # Home screen
    recipe/                  # Recipe display screen
  widgets/
    elio_progress_bar.dart   # Shared progress indicator
```
