6. Risks & known issues

  • Gemini first-attempt reliability — streaming sometimes fails on the first call after app launch and succeeds on retry. Hardened with multi-attempt retry, prewarm, JSON-truncation repair, lower temperature for onboarding. Not blocking, but a dedicated reliability pass is flagged.
  • APK size 72.3 MB — driven by mobile_scanner ML Kit. Likely needs Android App Bundles for Play Store distribution.
  • Cloud Functions backend not built — weeklyGenerations still client-writable. RevenueCat webhook and Gemini proxy with server-side rate limits are deferred. Acceptable for launch; revisit before scale.
  • iOS not built or tested yet — URL scheme placeholder must be filled, signing certificates set up. The biggest unknown unknowns sit here.
  • Hero illustration on onboarding screen 10 — placeholder pending Kate. Not a hard blocker, but worth landing before assets are submitted.
  • Palette tokens for freshness chips (freshGreen / perishThisWeek / perishToday) — placeholder hex pending Kate ratification.
